Welcome to GeLaTo
=================

|build-status| |docs| |binder|


**GeLaTo** is a python library for the Generalized Locally Toeplitz theory for IsoGeometric Analysis.

Installation
************

* **Standard mode**::

    python3 -m pip install .

* **Development mode**::

    python3 -m pip install --user -e .
